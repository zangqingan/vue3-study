一、vue3的基本认识

vue3是vue2版本的更新迭代,只是一些用法上不同,但是大体还是和vue2一样的。 它依然是渐进式的前端框架,它的特点,优势,主要内容和vue2版本类似基本不变。 Vue3 的两个核心功能依然是:声明式渲染、响应性。 声明式渲染:Vue 基于标准 HTML 拓展了一套模板语法,使得我们可以声明式地描述最终输出的 HTML 和 JavaScript 状态之间的关系。也就是SFC(单文件组件中编写) 响应性:Vue 会自动跟踪 JavaScript 状态并在其发生变化时响应式地更新 DOM。
